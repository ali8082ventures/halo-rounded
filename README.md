# Halo Rounded

Friendly rounded sans-serif display family by Glyphline / Catherine Bellot.
Variable font, wght 300–700. Licensed under the SIL Open Font License 1.1.

## Repo layout (Google Fonts upstream convention)
- fonts/variable/HaloRounded[wght].ttf — the variable font
- OFL.txt — license (SIL OFL 1.1, no Reserved Font Names)
- DESCRIPTION.en_us.html — catalog description

## HOW TO SUBMIT TO GOOGLE FONTS — step by step
1. Create a **public GitHub repo** named `halo-rounded` under your GitHub account
   (the copyright line says github.com/ali8082ventures/halo-rounded — 
2. Upload the contents of this folder to the repo. Tag a release, e.g. v1.000,
   and attach the TTF to the release.
3. Go to the Google Fonts issue tracker:
   https://github.com/google/fonts/issues/new/choose
   and pick **"Add font"**. Fill the template: family name "Halo Rounded",
   repo URL, license OFL, designer Catherine Bellot.
4. Google's onboarders will run their checks (fontbakery) and may request
   fixes; respond in the issue thread. Timeline: weeks to months.
5. Do NOT sell or list Halo Rounded anywhere else as a paid font from now on —
   OFL release is permanent and it must remain free.
